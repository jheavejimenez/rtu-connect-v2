This is a [Next.js](https://nextjs.org/) project bootstrapped
with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Setup

make sure you have nodejs installed on your machine and yarn installed globally
here is the link to install yarn https://classic.yarnpkg.com/en/docs/install/#windows-stable
link to install nodejs https://nodejs.org/en/download/

First, install the dependencies using yarn:

```bash
yarn install
```

## Getting Started

Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.
